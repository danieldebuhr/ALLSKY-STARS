# ALLSKY-STARS
Count stars from a given image. 

## Usage

    > allsky_stars.py -i <imageFile> [-s <schwellwert=150,0...255> -o <testOutputFile>]
  
    > allsky_stars.py -i sample_input_1.jpg -o sample_output_1.jpg
